# Arabic OCR - PDF to eBook Formats

A powerful web application built with Gradio that leverages Mistral AI's OCR capabilities to convert PDF documents into high-quality Markdown, EPUB, or MOBI formats.

## üöÄ Features

- **High-Quality OCR**: Uses Mistral AI's latest OCR model for accurate text extraction.
- **Multiple Export Formats**: Convert PDFs to Markdown, EPUB, or MOBI.
- **Embedded Images**: Handles images within documents by embedding them as base64 URIs.
- **Output Management**: View, download, or delete past conversion results directly from the interface.
- **Modern UI**: Clean and intuitive tabbed interface powered by Gradio.

## üìã Prerequisites

- Python 3.8 or higher
- [Pandoc](https://pandoc.org/installing.html) (Required for EPUB/MOBI conversion)
- A Mistral AI API Key

## üõ†Ô∏è Installation

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd gradioapp
   ```

2. **Install System Dependencies (Pandoc)**:
   This app requires Pandoc for EPUB/MOBI conversions.
   - **Windows/Mac/Linux**: [Download and install from the official website](https://pandoc.org/installing.html).
   - Verify installation by running `pandoc --version` in your terminal.

3. **Install Python dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Configure Environment Variables**:
   Edit the `.env` file located in the root directory and add your Mistral API key:
   ```env
   MISTRAL_API_KEY=your_api_key_here
   ```

## üìñ Usage

1. **Start the application**:
   ```bash
   python main.py
   ```
2. **Open the browser**:
   The app will typically be available at `http://127.0.0.1:7860`.

3. **Convert PDF**:
   - Upload your PDF file in the "Convert PDF" tab.
   - Choose your desired export format (Markdown, EPUB, or MOBI).
   - Click "Convert" and download your file once ready.

4. **Manage Outputs**:
   - Use the "Manage Outputs" tab to see history, preview content, or delete old files.

## üìÇ Project Structure

- `main.py`: Core application logic and Gradio interface.
- `requirements.txt`: Python package dependencies.
- `outputs/`: Directory where converted files are stored.
- `.env`: Environment configuration (API keys).
## üõë limitations
The EPUB files generated by this app might not upload directly to Kindle due to technical limitations. If that happens, just use a tool like [Calibre](https://calibre-ebook.com/)
 to convert the file into a Kindle-compatible EPUB.
